JavaScript String and Math Functions

String Object Functions

1. toUpperCase()

Converts all characters in a string to uppercase.

Example: "hello".toUpperCase()

2. substring()

Extracts a portion of a string between two specified indices.

Example: "JavaScript".substring(0, 4)

3. includes()

Checks if a string contains a specified value.

Example: "Hello, World!".includes("World")

Math Object Functions

1. Math.round()

Rounds a number to the nearest integer.

Example: Math.round(4.6)

2. Math.random()

Generates a random number between 0 (inclusive) and 1 (exclusive).

Example: Math.random()

3. Math.max()

Returns the largest of the given numbers.

Example: Math.max(10, 20, 30)
